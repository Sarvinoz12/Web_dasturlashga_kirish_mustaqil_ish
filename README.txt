Razzaqberdiyeva Sarvinozning Web dasturlashga kirish fanidan mustaqil ishi.

Fayl linki: 

GitHub dan proektni o'zimizga yuklab olib ishlatamiz
buning uchun Xampp yoki open server ishlab turishi kerak.
 
Bazaga ma'lumot qo'shish yoki o'chirish uchun sizda quydagi bazalar bo'lishi kerak

Baza nomi:dizayn

Ushbu baza ichida siz:project,user,xabar nomli jadvallarni ham yaratishingiz kerak.

project jadvali:id,nomi,rasmi,matni nomli ustunlardan iborat bo`lishi kerak.

user jadvali:is,ism,password,login,status nomli ustunlardan iborat bo`lsin.

xabar jadvali:id,ism,pochta,telefon,xabar nomli ustunlardan iborat.

Brauzerga localhost/sarinoz.com/ deb izlanadi shunda saytimiz ko`rinadi.

Saytimizga yangi proekt qo`shish uchun localhost/sarvinoz.com/create_project.php bo`limga kiriladi va yangi proekt qo`shish tugmasi bosiladi va proekt qo`shiladi.

Saytimizda User qo`shish uchun localhost/sarvinoz.com/user.php bo`limga kiriladi.

Va ushbu bo`limdan user qo`shish tugmasi bosiladi va bizda user qo`shish oynasi ko`rinadi.

BU bo`limda bizga  Ism,password,login so`raladi bu so`rov to`ldiriladi keyin save tugmasi bosiladi,va shu bilan yangi user yaratiladi.

Mijozlarimiz biz bilan bog`lanishi uchun bizga bog`lanish oynasi berilgan.bu oyna to`ldiriladi va yuborish tugmasi bosiladi.keyingi oynada xabar bazaga uzatiladi va saqlandi yozuvi ko`rinadi.

Biz xabarni baza orqali yoki localhost/sarvinoz.com/xabarlar.php bo`limidagi xabarlarni ko`rish tugmasi orqali ham ko`rishimiz mumkin.

Saytimizda sizga web dizayn haqida ma`lumot beriladi.

Saytimizda siz mijozlarimizning biz haqimizdagi fikrlari bilan ham tanishishingiz mumkin.
